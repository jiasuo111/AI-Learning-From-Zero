## 1. 需要安装小皮面板

> [小皮面板(phpstudy) - 让天下没有难配的服务器环境！](https://www.xp.cn/)

正如标题所示，该软件是专门配置服务器的一款集成软件。我们现在只需要用到里面的 MySQL 功能。

### ⚠️ 给新手的忠告

**新手建议不要从官网下载 MySQL**，非常麻烦。并且一旦没有安装成功（假如各种因素影响，第一次失败了），那一个人真的很难清理干净并且重新操作，会出现非常多的问题。

**假如已经有这款软件的人，也绝对不要删除**，非常难以卸载，更难以再装回去。

**新手建议：使用这款软件，一键配置 MySQL。**

只需要一路点下载就好了。

---

## 2. 给新手的一句忠告

> **将所有软件都下载在英文目录中，路径中不能存在中文，否则会为你的未来产生难以想象的 bug，并且你几乎很难找得出来。**

可以给一点建议：将所有代码有关的软件，全部放在 `software` 文件夹里面。例如： D:\software


这样做会让你的根目录**赏心悦目**，有助于你以后的文件管理和寻找。因为以后需要你自己配置环境、没办法一键完成的时候，需要你手动找到他的 exe 文件。

---

## 3. 安装 MySQL 5.7 版本

**因为大部分企业都是这个版本，5 以上将要收费。**

1. 点开小皮面板里面的**软件管理**，点击下载 MySQL
2. 点开**首页**，将 MySQL 启动

**这里就配置完成了，就是如此简单。**

---

## 4. 安装 PyCharm

> PyCharm，您需要的唯一 Python IDE  
> 官网：https://www.jetbrains.com/zh-cn/pycharm/

可能会有人问我：为什么不选择 MySQL 对应的特定软件 DataGrip？

因为 PyCharm 的功能 DataGrip 都能做到，并且后面学 Python 也可以用 PyCharm，所以我在这里选择这款软件教你们。

**可以下载专业版本**，可以在某宝花几块钱买一个激活码，会有全程的操作教程如何激活，很简单。

---

## 5. 安装 Python

> 官网：Welcome to Python.org

**一定不要安装最新的版本**，会有很多问题。可以安装低一等级的版本，或者可以问 AI 目前哪个版本比较稳定。

### ⚠️ 切记

一定要选择**将环境配置进 PATH 环境里面**，不然后面安装各种插件将非常麻烦，不能一步解决。

**这很重要，不要点太快，注意看！**
